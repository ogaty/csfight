c#のstring format

こいつは覚えておくと便利。sprintfみたいな感じですね。

0埋めする

return String.Format("{0:0000}/{1:00}/{2:00} {3:00}:{4:00}:{5:00}", year, month, day, hour, minute, second);

こんな感じで書けば0埋めして返してくれます。

number_format

数字を３桁区切りにするような感じ。

return String.Format("{0:N0}", price);

小数点

00.00と書くと小数点2桁になります。

return String.Format("{0:00.00}", rate);
