新しいウィンドウを開く

単純にstoryboardを開き、controlを押しながらボタンを対象のウィンドウにドラッグさせるだけです。
segueと呼ばれています。

segueで遷移した新しいウィンドウにパラメータを渡す

遷移元のViewControllerでprepareForSegueってのをオーバーライドさせましょう。
んで遷移先のメンバ変数に値をセットすればOKです。
public override void PrepareForSegue(NSStoryboardSegue segue, NSObject sender)
                    {
                        ((SearchLatLngViewController)segue.DestinationController).inputPlace = UserPlace.StringValue;
                        base.PrepareForSegue(segue, sender);
                    }
