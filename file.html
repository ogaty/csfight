ファイル操作

wpfの場合

パス制約がないのでStreamReaderでさくっといけます。
                    string html = "";
                    using (StreamReader reader = new StreamReader(path)) {
                        html = reader.ReadToEnd();
                    }

uwpの場合

async/awaitを使うことが必須です。
                    var data = await systemFolder.GetFileAsync("file.txt");
こんな感じ。受け取る型もStorageFile型なので読み込みもめんどい。

Macの場合

こちらはuwpみたいにしなくてもStreamReaderでOK。
ただしストア用アプリにするなら変なパスを指定しないように。
                    string html = "";
                    using (StreamReader reader = new StreamReader(path)) {
                        html = reader.ReadToEnd();
                    }
