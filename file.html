ファイル操作

Fileクラス

基本的にOpenで読み込み、Closeで閉じます。
fopen、fcloseと変わらない感じ。
FileModeオプションは下記のような感じですね。
Open: 普通に開く、ファイルがないとエラー
OpenOrCreate: ファイルが無いと生成して開く
Create: 生成する、ファイルがあると上書き
CreateNew: 生成する、ファイルがあるとエラー
Append: 追記、ファイルが無いとエラー
Truncate: ファイルを空にする、ファイルがないとエラー

wpfの場合

パス制約がないのでStreamReaderでさくっといけます。
string html = "";
                            using (StreamReader reader = new StreamReader(path)) {
                                html = reader.ReadToEnd();
                            }

uwpの場合

async/awaitを使うことが必須です。
var data = await systemFolder.GetFileAsync("file.txt");
こんな感じ。受け取る型もStorageFile型なので読み込みもめんどい。

Macの場合

こちらはuwpみたいにしなくてもStreamReaderでOK。
ただしストア用アプリにするなら変なパスを指定しないように。
string html = "";
                            using (StreamReader reader = new StreamReader(path)) {
                                html = reader.ReadToEnd();
                            }
